## üöÄ <sub>Getting started</sub>
<details>
<summary>Click to reveal</summary>

---
Install the dependencies...

### `npm install`

...then start [Rollup](https://rollupjs.org):

### `npm run dev`

Navigate to [localhost:5000/emuos](http://localhost:5000/emuos). You should see your app running. Edit a component file in `src`, save it, and hot reload will refresh the page automatically.

By default, the server will only respond to requests from localhost.

## <sub>Building and running in production mode</sub>

To create an optimised version of the app:

### `npm run build`
